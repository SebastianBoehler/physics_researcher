# Code of Conduct

This project follows the Contributor Covenant.

## Our Standards

- Be respectful and constructive.
- Assume good intent, but focus feedback on the code and the problem.
- Avoid harassment, discrimination, or personal attacks.
- Keep technical disagreement factual and specific.

## Enforcement

Project maintainers may remove comments, issues, or contributions that violate these expectations.

## Reporting

Report conduct concerns privately through a GitHub security advisory or direct maintainer contact.
